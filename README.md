# CodeAlpha_DiseasePrediction

## Task 4 - Disease Prediction from Medical Data

This project predicts whether a tumor is malignant or benign using the Breast Cancer dataset.

### Approach
- Classification algorithms used: Logistic Regression, Random Forest, SVM
- Feature scaling applied using StandardScaler
- Model evaluated using Accuracy, Precision, Recall, F1-Score, and ROC-AUC

### Files
- `disease_prediction.py` - Main script for training and evaluation
- `disease_prediction_model.pkl` - Saved best-performing model
- `scaler.pkl` - Saved feature scaler
- `class_distribution.png`, `confusion_matrix.png`, `feature_importance.png`, `model_comparison.png`, `roc_curve.png` - Generated visualizations

### Tech Stack
Python, Scikit-learn, Pandas, Matplotlib, Seaborn
